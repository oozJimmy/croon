# croon
Python CLI tool that shows local lunar and solar data in a neofetch-like display. Includes moon art!

### Third-party Python Libraries
+ [Rich](https://github.com/Textualize/rich)
+ [Requests](https://github.com/psf/requests)

### Data API
+ [US Navy Astronomical Applications Department](https://aa.usno.navy.mil/data/index)
